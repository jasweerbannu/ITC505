Addendum

  • Game Idea: I created a Lights Out game where players turn off lights on a 5x5 grid by clicking tiles.
  • Layout: The HTML structure includes a start screen with a button to begin the game, making it easy for users to understand.
  • Visual Design: I chose bright colors for the tiles and a dark background to make the game look good and the lights easy to see.
  • Responsive Design: I used CSS to ensure the game looks good on different screen sizes, making it accessible for all users.
  • Game Logic: I wrote JavaScript to control how the game works, like generating patterns and toggling lights when tiles are clicked.
  • Move Counter: I added a counter to show how many moves the player has made, giving them feedback on their progress.
  • Win Condition: When all lights are off, a pop-up message appears to let the player know they won the game.
  • Challenges: I faced issues with getting the tiles to display in a grid, which I fixed using CSS Flexbox for a better layout.
  • Debugging: I improved my debugging skills by finding and fixing problems in the code that caused the game to misbehave.
  • Learning: This project taught me the importance of planning, user experience, and problem-solving in game development.
